# Equality NJ PAC - Landing Page

A modern, mobile-responsive static landing page for a Political Action Committee focused on equality in New Jersey.

## Features

- **Responsive Design**: Optimized for both desktop and mobile devices
- **Signup Form**: Fully validated form with real-time error messages
- **Modern Styling**: Built with Tailwind CSS for a sleek, professional appearance
- **GitHub Pages Ready**: Static HTML file ready for hosting

## Form Validation

The signup form includes validation for:
- First Name (minimum 2 characters)
- Last Name (minimum 2 characters)
- Email Address (HTML5 email validation)
- Zip Code (5-digit US zip code)

## Local Development

To view the page locally:

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000/index.html in your browser.

## Deployment

This site is designed to be hosted on GitHub Pages. Simply enable GitHub Pages in your repository settings and point it to the main branch.
